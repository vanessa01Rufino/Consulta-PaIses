Consulta de Países

Este projeto foi desenvolvido como trabalho final da disciplina de Programação para Internet, do curso de Gestão da Tecnologia da Informação (GTI) – 4º semestre da Fatec Itu Dom Amaury Castanho.

A aplicação permite consultar informações sobre países de forma simples e interativa, utilizando uma API pública para obter dados atualizados.
Funcionalidades

    Busca de países por nome

    Filtro de países por continente

    Exibição de informações como:
        Nome do país
        Capital
        População
        Moeda

    Visualização de detalhes ao clicar em um país

    Histórico de buscas armazenado no navegador

    Paginação dos resultados

Tecnologias utilizadas

    HTML5
    CSS3
    JavaScript
    API REST Countries
    SweetAlert2

Objetivo

O objetivo do projeto é aplicar conceitos de desenvolvimento web, consumo de APIs, manipulação de DOM e armazenamento local (LocalStorage), proporcionando uma experiência interativa ao usuário.
